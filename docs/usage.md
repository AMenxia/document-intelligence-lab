# Usage

## Start the app

```bash
streamlit run app/streamlit_app.py
```

## Recommended workflow

1. Upload PDFs.
2. Select template and analysis mode.
3. Run Docling.
4. Review detected items.
5. Correct selected tables.
6. Review visual items and add optional notes.
7. Run VLM on selected visuals.
8. Build the LLM feed.
9. Run extraction.
10. Preview and download Excel outputs.
